# LinguaCards - Language Learning Flashcard App

A modern, interactive flashcard application for language learning with spaced repetition system, Excel/CSV upload, and user authentication.

## üåü Features

### Core Features
- **Interactive Flashcards**: Flip cards with smooth animations
- **Spaced Repetition System**: Intelligent review scheduling (1, 3, 7, 14, 30 days)
- **Bulk Import**: Upload vocabulary lists via Excel/CSV files
- **User Authentication**: Secure login/registration system
- **Progress Tracking**: Dashboard with learning statistics
- **Responsive Design**: Works on mobile, tablet, and desktop

### Advanced Features
- **Excel/CSV Upload**: Import hundreds of cards at once
- **File Validation**: Automatic validation with error highlighting
- **Pronunciation**: Text-to-speech for word pronunciation
- **Card Types**: Color-coded cards (nouns, verbs, adjectives)
- **Search & Filter**: Find cards by language, type, or status
- **Review Queue**: Cards due for review are highlighted
